Sample Menu Page!

published: 12.11.2019, 12:50

Recently, I thought it would be fun to create a sample menu page for one of my favorite restaurants. I incorporated some animation using CSS to make the menu more interactive for the user and create a better UX. It was pretty fun!

Bourbons is an amazing restaurant located in Framingham, MA. The young owner, Luke Folley, has managed to build a gem of a restaurant in Framingham. Visitors can expect creative, tasty cocktails, usually in tune with the seasons. And the food is equally as good as the drinks. Bourbons has become a local hotspot for its service, quailty, atmosphere, and good Americana food. Definitely check it out if you're ever in the area!

Partying Robot!

published: 12.11.2019, 12:53

I've recently finished General Assembly's free tutorials. One of my favorite projects was making a partying robot! Using CSS animation, your client, an eccentric millionaire named Cotter, hires you to build a robot that he can take with him when he goes partying.

You can click three different buttons. One toggles some laser eyes for the robot. The other toggles Cotter's dancing (so he can take a break). The last button changes the background color. Repeatedly pressing it creates a strobe effect! Click to try it out!

Also, feel free to check out GA's tutorials at Dash!. They were really helpful, easy to follow, and a great starting point to build my CSS & JavaScript skills with.
